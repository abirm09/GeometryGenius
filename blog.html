Questions Answers

1. What is DOM and what is the purpose of DOM?

DOM's full meaning is Document Object Model.When a webpage is loaded, browser js engine create DOM .Then javascript gets power to create, remove, modify html elements.And javascript did this dynamically.

  • Javascript DOM can create HTML element.
  • Javascript DOM can change HTML element.
  • Javascript DOM can add CSS.
  • Javascript DOM can create HTML events.
  • Javascript can do many more things.

2. How will you select HTML elements using DOM? Name the DOM methods.

There are many ways to select HTML elements.Some of them are:

  • document.getElementById("id-name");
  • document.getElementsByClassName("class-name");
  • document.getElementsByTagName("tag-name");
  • document.querySelector (".class-name or #id-name");
  • document.querySelectorAll(".class-name or tag-name");
  • etc.

3. What is event bubble?

In event bubbling, if we add click event on an element the event can travel up from the triggered element to its parent element. It's also call event propagation.We can prevent this propagation by stopPropagation() method.

4. What is a callback function and why will you use it?

If a function receive a function as an argument that call's a callback function.A callback function is used to execute code in response to an event.
